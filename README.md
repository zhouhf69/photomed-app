# PhotoMed 医疗影像管理平台

一个现代化的医疗影像管理平台，提供安全、高效的影像存储和管理服务。

## 功能特性

- **安全**：企业级数据保护
- **快速**：高效影像处理
- **智能**：AI 辅助诊断

## 技术栈

- React 18
- Vite
- Tailwind CSS
- React Router

## 开始使用

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 部署

本项目已配置为自动部署到 Vercel。
